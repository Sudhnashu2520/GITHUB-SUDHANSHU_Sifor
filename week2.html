What is CSS? What are different type of selectors in CSS?

CSS stands for cascading style sheet, it is used to style our webpages. It provides inbuilt classes to style web pages. There are 3 types of CSS : 1.inline, 2.internal and 3.External. Selectors are used by css to give styling to a particular element. if element we have to style is provided by unique id then we can assess that id with an '#''.It will style to unique element containing that id
if element we have to style is provided by class then we can assess that id with an '.'.It will style to element containing that classname
if element we have to style is an element then we can directly use element name. It will style whole p tags in that html page
e.g. p{ color:"red" }

What is CSS boxmodel?

Css boxmodel is visual representation how to style a component. It is like a container consist of muliple elements which as attributes border, margin ,height,width etc. by which we can style the container as per our need
